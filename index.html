因为有美丽的错过，才会有刻骨铭心的回忆

发表于 2019-06-25 | 分类于 文章

　　世间万物，唯草木与山水是我心之所依，亦是我此生之所爱。我总想着，凡事但尽全力，想到即去做，不负岁月，亦不辜负每一次的机遇，就不会心生怨悔。奈何世事造化弄人，人情冷暖亦是倏忽变换，无论我如何的珍惜当下的每一刻，总还是会留下遗憾，终是会错过许多唾手可得的机遇，乃至错过许多的人和事。
　　有人说，这世间最美的，当是相遇。遇见一个人，的确是一种幸运。但所遇见之人，未必是能与你同舟共济之人。也许有情者未必缘深，缘深者也未必与你有缘。而你所苦苦追寻与等待的人，与你有缘相识，却未必能携手白头;那个看似你生命中的归人，值得托付一生，却浑然不知他本是你生命中的过客。
　　“我不是归人，是个过客，”我们每个人又何尝不是如此呢?在自己得人生戏剧里充当着行色匆匆的路人，又在他人的故事里充当着过客。在别人的故事里，永远流着属于自己的泪。

岁月安好，静守流年

发表于 2019-06-23 | 更新于 2019-06-25 | 分类于 文章

　　时间转动着岁月的沙盘，点滴回忆被搁浅在了狭小的角落里，零零碎碎的记忆碎片摩梭着我们的认知。不知何时想起时，却已是此去经年，空守城池。
　　如今回想，已不是年少那般懵懂无知的脾性了。多多少少被磨去了那与世争锋的棱角。曾经的头破血流也已换得如今的万贯江湖，独我一人浪迹天涯。一思茶，思终缘，一渡酒，渡流年。
　　繁华若水三千余，唯取一瓢不改颜，此生桃花万籁开，不剩芬芳一沥红。曾与你前世相守，不枉今生相逢，也许是命中自有意，时光不赖，让你我不终不散。时儿三许，时儿易散。

如果注定要一个人走

发表于 2019-06-18 | 更新于 2019-06-25 | 分类于 文章

　　并不是所有的人都在爱情之中，或许，这个情人节，注定要一个人走，哪怕你从未停止过追爱的脚步。　　
　　毋庸置疑，爱情是一个正常人的美好向往。但我们往往还在等爱的路上，或者在追爱的路上、刚刚失去爱的路上，没有爱情的日子，终究是存在。就说，这个情人节，你会是一个人过吗？

　　

　　如果，注定要一个人度过情人节，你的心情怎么样呢。你大概会独自伤悲，看着路边牵手走过的情侣，背过脸，偷偷抹一把眼泪。我要奉劝你：即便如此，你一定不要随便找一个人牵手，你甚至要勇敢地说，“我单身，我骄傲”。相比那些找到爱情的人，你有更多选择爱谁的权利，不是吗？
